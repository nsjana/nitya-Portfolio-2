# nitya-Portfolio-2 — Next.js Starter

This repo is a Next.js (App Router) port of the static portfolio template.

Defaults used:
- JavaScript (no TypeScript)
- Next.js app router (app/)
- Global CSS (app/globals.css)

How to run locally
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Open `http://localhost:3000`

Notes
- Replace placeholder text (Your Name, links, projects) in `app/page.jsx` and components.
- The contact form uses Formspree; set your Formspree form URL in the form `action`.

Deploying
- Recommended: deploy to Vercel (connect the GitHub repo and it will detect Next.js).
